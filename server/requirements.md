
** Server Requirements **

*** 프로젝트 개요 ***
- 쇼핑몰용 서버
- 다국어 지원 (일본어, 영어)

*** 코드스타일 ***
- ES Module 사용 
- TypeScript 를 사용하지 말고 JavaScript 만 사용 

*** 스택 ***
- Express JS 사용 
- 파일 업로드 처리로 Multer 사용 
- 기본 DB 로 MongoDB 사용 

*** DB 모델 *** 
- User: 회원 모델
- Order: 주문 모델 
- Product: 상품 모델 
- Cart: 장바구니 모델
- Ticket: 고객 CS 처리용 모델
