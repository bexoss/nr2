# 라우팅/URL 규칙

## 패턴
- 소문자 + 하이픈(`kebab-case`), 트레일링 슬래시 미사용(`/path`).
- 컬렉션/상세: `/posts` → `/posts/<slug>`.
- 검색/필터는 쿼리(`?q=...&tag=...`).

## 상태 코드
- 영구 이동 301, 임시 302. 삭제는 410 고려.
- 존재하지 않음은 404, 사용자 친화적 복귀 링크 제공.

## i18n(선택)
- 언어 프리픽스: `/ko/...`, `/en/...`.
- 기본 언어는 루트(`/`) 허용, canonical로 언어별 중복 방지.

- [ ] 중복/파편화 없는가?
