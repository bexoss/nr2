# 성능 가이드 & 예산

## 목표(Core Web Vitals)
- LCP ≤ 2.5s, CLS ≤ 0.1, INP ≤ 200ms.

## 예산(초기 페이지)
- JS 전송 ≤ 150KB gzip, CSS ≤ 80KB gzip.
- 이미지 개별 ≤ 200KB, 폰트 파일 ≤ 100KB.

## 원칙
- 이미지: 적응형 사이즈/포맷(WebP/AVIF), 지연 로드.
- 폰트: 서브셋, `font-display: swap`, 가변폰트 선호.
- JS: 지연/조건 로딩, 중복 의존성 제거, 서버 렌더 우선.
- 캐시: 정적 에셋 해시, 장기 캐싱, HTTP/2 우선.

## 체크리스트
- [ ] 초기 렌더에 불필요한 JS 배제
- [ ] 이미지/폰트 최적화
- [ ] 코드 분할/지연 로딩 적용
- [ ] Lighthouse 90+ 유지
